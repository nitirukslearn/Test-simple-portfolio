# Data Analyst Portfolio - Nitiruk

A professional, modern, and responsive portfolio website designed for a Data Analyst. Built with a focus on clean typography, a dark-themed aesthetic, and smooth user interactions.

## 🚀 Features
- **Modern Dark Theme:** A sleek, high-contrast UI designed for readability.
- **Responsive Design:** Optimized for various devices, from desktops to mobile phones.
- **JetBrains Mono Typography:** Utilizes professional developer-centric fonts.
- **Project Showcase:** Dedicated section to highlight key data analysis projects.
- **Smooth Navigation:** Interactive scrolling and UI elements for an enhanced user experience.

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **Vanilla CSS:** Custom styling with Flexbox and Grid.
- **JavaScript:** Client-side interactivity and smooth navigation.
- **Google Fonts:** JetBrains Mono for a technical aesthetic.

## 📂 Project Structure
- `index.html`: Main document structure.
- `style.css`: Visual styling and layout.
- `script.js`: Interactive logic and navigation.
- `plan/`: Project documentation and implementation roadmap.

## 🌐 Deployment
The project is designed to be hosted on **GitHub Pages**.
